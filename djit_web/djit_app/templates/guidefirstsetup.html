First Setup Guide

1Create account

first of all you need to create account at site

2setup ssh key

You must create your own pair of public and private ssh keys

Then you need to put public ssh key in your profile

This action will create account and directory for you in remote server

3Create repository

You can create repository using site, then connect to remote with ssh

You can connect to remote, using git bash, terminal, your IDEs tools - anything you want

If success - you can do any git actions - commit, push etc

4Place your bare repository

You can place existed repository at remote, using upload form at repo creation
